# Source Wallet

This **demo** is intended solely for demonstration purposes. Joint **team development** of a crypto wallet designed to store BTC. 

- It is executed on **UIKit**, according to the **MVC** structure with the maximum possible division of the code into **sub-parts**. 
- A comfortable and pleasant design was selected for user experiences, **partially implemented programmatically**. 
- Source Wallet has nothing to do with Valve corporation, it's just a reference :)  
- The project consists of **many screens**, only some of them will be shown below.
